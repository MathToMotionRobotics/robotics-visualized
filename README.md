# From Math to Motion

> Learn robotics from first principles.

From Math to Motion is an open visual robotics education project.

We explain how mathematical ideas become physical robot motion through:

- visual intuition
- mathematics
- simulation
- executable code
- real hardware experiments

## Current Milestone

Produce three pilot episodes and establish a repeatable content workflow.

## Teaching Method

Each episode follows the same structure:

1. Start with a visible robotics problem
2. Build physical intuition
3. Introduce only the necessary mathematics
4. Verify the idea in simulation
5. Test it on real hardware
6. Explain what fails and why
